# 🎬 YouTube Clone Project

A professional front-end YouTube homepage clone built using HTML and CSS.

---

## 🚀 Features

- 🔍 Navigation bar with search functionality  
- 📂 Sidebar with multiple sections:
  - Home  
  - Trending  
  - Subscriptions  
  - Library  
  - History  
  - Watch Later  
- 🎥 Video grid layout with thumbnails  
- 👤 Channel name, views, and time display  
- 🎨 Clean and modern UI design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  

---

## 📁 Project Structure

```
youtube-clone/
│── index.html
│── style.css
```

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser
